Remarks
Use this inteface for child components of classes implementing IProjectComponent when they have absolute position on the task editor. When inserting a deserialized sequence, the project components position must be adjusted to be aligned with the insert point. When a project component contains further children with absolute position (not relative to the component itself), their position also have to be adjusted. The AdjustPosition method is called by the insert command with the correct adjustments.
